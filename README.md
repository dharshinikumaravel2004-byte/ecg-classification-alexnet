ECG Classification using AlexNet

Overview

This project implements an AlexNet-based deep learning model for ECG signal classification. The model is trained on the MIT-BIH Arrhythmia dataset to classify different types of heartbeats.

Dataset

- MIT-BIH Arrhythmia Dataset
- Preprocessing includes normalization and reshaping of ECG signals for AlexNet input.

Model

- Architecture: AlexNet
- Framework: TensorFlow / Keras
- Task: Multi-class ECG heartbeat classification

Performance Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Matthews Correlation Coefficient (MCC)

Project Structure

- notebooks/ : Training and evaluation notebooks
- results/ : Performance results and plots
- models/ : Saved model files
- README.md : Project documentation

Requirements

- Python
- TensorFlow
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
